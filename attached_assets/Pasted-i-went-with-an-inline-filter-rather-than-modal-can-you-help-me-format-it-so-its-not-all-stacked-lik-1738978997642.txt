Inline Filter Formatting
Reasoned for 54 seconds
Below is a simple inline filter bar example that places your “All Statuses” select, the two date fields, and your “Clear” (or “Apply”) button(s) on a single horizontal row—rather than stacking vertically. You can adapt this approach to your own layout.

1) Example HTML
html
Copy
<div class="filter-bar d-flex align-items-center gap-2 mb-3">
  <!-- Status Dropdown -->
  <select id="statusFilter" class="form-select w-auto" aria-label="Status Filter">
    <option value="">All Statuses</option>
    <option value="pending">Pending</option>
    <option value="processing">Processing</option>
    <option value="completed">Completed</option>
  </select>

  <!-- Start Date -->
  <input type="date" id="startDate" 
         class="form-control w-auto" 
         placeholder="mm/dd/yyyy" 
         aria-label="Start Date">

  <!-- End Date -->
  <input type="date" id="endDate" 
         class="form-control w-auto"
         placeholder="mm/dd/yyyy" 
         aria-label="End Date">

  <!-- Apply or Filter Button (optional) -->
  <button class="btn btn-primary" type="button" onclick="applyFilters()">
    Apply
  </button>

  <!-- Clear Button -->
  <button class="btn btn-secondary" type="button" onclick="clearFilters()">
    Clear
  </button>
</div>
Notes
.d-flex align-items-center gap-2 (Bootstrap utility classes) places everything in one row with a small gap.
.w-auto on the select and inputs keeps them from stretching to 100% width.
You can optionally keep or remove labels—this snippet uses placeholders (mm/dd/yyyy) and the select’s text for clarity.
2) Matching CSS (If Not Using Bootstrap Utilities)
If you’re not using Bootstrap or similar, just replicate those utility classes yourself:

css
Copy
.filter-bar {
  display: flex;
  align-items: center;  /* vertically center items */
  gap: 0.5rem;          /* spacing between items */
  margin-bottom: 1rem;  /* optional for spacing below */
}

/* Let them size themselves rather than full width */
.filter-bar .form-select,
.filter-bar .form-control {
  width: auto; 
  display: inline-block; 
  vertical-align: middle;
}

/* Additional styling to your liking */
.filter-bar .btn {
  height: auto;       /* Let the button match input height */
  font-size: 0.9rem;  /* Example smaller text */
  padding: 0.45rem 0.75rem;
}
3) Result
One tidy row with a status dropdown, two date inputs, and a Clear/Apply button.
On small screens, you may want them to wrap. If so, add flex-wrap: wrap; to .filter-bar.
With these tweaks, your filter controls will no longer appear stacked. You get a clean inline filter bar at the top of your page. Enjoy!
