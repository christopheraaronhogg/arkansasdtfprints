function showFullSizePreview(src) {
    const modal = document.getElementById('previewModal');
    const modalImg = document.getElementById('modalImage');
    modal.style.display = 'flex';
    modalImg.src = src;

    modal.onclick = function(e) {
        if (e.target === modal || e.target.className === 'close-modal') {
            modal.style.display = 'none';
        }
    };
}
</script>

2. Suggested CSS Adjustments
Below is a minimal override. You’ll keep your existing button styles and .size-inputs card styles (so they have a border/shadow). Then we remove the “big box” look from the top section.

css
Copy
/* Remove the boxy container around order details */
.order-details {
  background: none;       /* or transparent */
  box-shadow: none;       /* remove shadow if it was there */
  border: none;           /* remove any border */
  padding: 0;             /* you can use margin/padding on .row or .col instead */
}

/* Make sure the row itself has some space */
.order-details .row {
  margin: 0;
  padding: 1rem 0; /* optional, just a bit of space above/below */
}

/* The item cards remain the same or slightly more prominent */
.size-inputs {
  background: #fff;
  border: 1px solid var(--border-color);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
  transition: transform 0.2s ease;
}

.size-inputs:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-md);
}

/* Extra margin between the items heading and order details row */
.section-heading {
  margin-top: 2rem; /* if you want a clear gap */
}

/* If you want to lighten the overall page background slightly for contrast */
body {
  background-color: #f1f5f9; /* a shade darker than pure white */
}
Notes:
With the background: none; on .order-details, you effectively drop the “container behind the order info.”
If you still see a border in your screenshot, ensure you remove any previous references to border: 1px solid ... or border-radius from .order-details-area or .order-info.
Adjust spacing (margin, padding) to get the exact look you want.
Result
The Order Information top section now appears as simple text on your main page background, with just enough spacing to keep it readable.
The Order Items remain in visually distinct “cards” that have a border and subtle shadow.
Additional whitespace (margin) around headings and columns provides a cleaner, more open feel.
Feel free to adjust the variables, shadow intensities, or spacing to best suit your style!
