function showFullSizePreview(src) {
  const modal = document.getElementById('previewModal');
  const modalImg = document.getElementById('modalImage');
  modal.style.display = 'flex';
  modalImg.src = src;

  modal.onclick = function(e) {
    if (e.target === modal || e.target.className === 'close-modal') {
      modal.style.display = 'none';
    }
  };
}

2) Matching CSS
Below is key CSS to ensure:

The 4 info cards line up in a single row on larger screens, with minimal gaps and a consistent card style.
Status card is small, matching the same style.
Select has reduced padding and a smaller font.
Item cards (.size-inputs) match the same card style as .info-card.
css
Copy
/************************************
  Shared Card Styles
************************************/
.info-card,
.size-inputs,
.status-card {
  background: var(--surface-color);
  border: 1px solid var(--border-color);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
}

/************************************
  Info Cards Grid
************************************/
/* 4 columns with minimal gap 
   (they'll wrap on small screens) */
.info-cards-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 1rem;
}

/* Info Card layout */
.info-card {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  padding: 1.25rem;
  transition: transform 0.2s ease;
  text-align: center;
}

/* Slight hover effect */
.info-card:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-md);
}

/* Icons/labels/values in each card */
.card-icon {
  font-size: 1.6rem;
  color: var(--primary-color);
  margin-bottom: 0.5rem;
}
.card-icon.text-success {
  color: var(--success-color);
}
.info-label {
  font-size: 0.85rem;
  color: var(--text-secondary);
  margin-bottom: 0.25rem;
}
.info-value {
  font-size: 1rem;
  font-weight: 600;
  text-align: center;
  white-space: normal;
  overflow-wrap: break-word; 
}

/* On smaller screens, wrap to fewer columns */
@media (max-width: 992px) {
  .info-cards-grid {
    grid-template-columns: repeat(auto-fit, minmax(180px, 1fr));
  }
}
@media (max-width: 576px) {
  .info-cards-grid {
    grid-template-columns: 1fr;
  }

/************************************

Explanation
.info-cards-grid

grid-template-columns: repeat(4, 1fr) ensures 4 columns on wide screens.
On mid-size screens (max-width: 992px), it uses repeat(auto-fit, minmax(180px, 1fr)) so it might show 2 or 3 cards per row depending on available width.
On small phones (max-width: 576px), it goes to 1 column.
Tight Gaps

We used gap: 1rem; so the space between cards is minimal.
The same gap: 1rem; for your .images-grid, so the item cards match visually.
Matching Card Styles

.info-card, .size-inputs, .status-card share the same background, border, radius, and shadow.
This ties them together visually, rather than having the info cards look drastically different from the item cards.
Simplified Select

.status-select.form-select uses padding: 0.45rem 1rem; font-size: 0.9rem; so it’s smaller than the default Bootstrap style.
We keep the .submit-btn style consistent.
This should yield a tighter, more uniform layout with minimal gaps, better matching item cards, and a nicer status dropdown. Make sure to remove any old leftover CSS that might be forcing large margins or vertical stacking. Then you’ll have a cohesive, compact design!
