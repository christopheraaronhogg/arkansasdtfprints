function showFullSizePreview(src) {
  const modal = document.getElementById('previewModal');
  const modalImg = document.getElementById('modalImage');
  modal.style.display = 'flex';
  modalImg.src = src;

  modal.onclick = function(e) {
    if (e.target === modal || e.target.className === 'close-modal') {
      modal.style.display = 'none';
    }
  };
}

2) Essential CSS
Below are key rules to ensure the four info cards line up horizontally, the email can wrap, and the status form looks better. Remove or override any old .order-info or .order-information rules that might force vertical stacking!

css
Copy
/************************************
  Info Cards Row
************************************/
/* Flex container for the 4 boxes. 
   .d-flex, .flex-wrap, and .gap-3 come from Bootstrap utility classes. */
.info-cards-row {
  /* You can add justify-content: space-between; or center, etc. if needed. */
}

/* Each box is 200x200 on large screens. 
   Adjust if you want them bigger or smaller. */
.info-card {
  background: var(--surface-color);
  border: 1px solid var(--border-color);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  width: 200px;
  height: 200px;
  text-align: center;
  padding: 1rem;
  transition: transform 0.2s ease;
}

/* On hover, a slight lift */
.info-card:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-md);
}

/* Icon at top */
.card-icon {
  font-size: 1.8rem;
  color: var(--primary-color);
  margin-bottom: 0.5rem;
}
.card-icon.text-success {
  color: var(--success-color);
}

/* Labels & values in each card.
   Force wrapping on emails with overflow-wrap. */
.info-label {
  font-size: 0.85rem;
  color: var(--text-secondary);
  margin-bottom: 0.25rem;
}
.info-value {
  font-size: 1rem;
  font-weight: 600;
  white-space: normal;            /* allow multiple lines */
  overflow-wrap: break-word;      /* break long lines if needed */
  word-wrap: break-word;          /* older fallback */
}

/************************************
  Status Card
************************************/
.status-card {
  background: var(--surface-color);
  border: 1px solid var(--border-color);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
  /* Just a single row below the info cards for the form. 
     Already .d-flex, .align-items-center, .gap-3 via markup. */
}

/************************************
  Responsive
************************************/
/* On smaller screens (<992px), let them wrap 2 per row. */
@media (max-width: 992px) {
  .info-card {
    width: calc(50% - 1rem);
    height: 220px; /* keep them somewhat square-ish */
  }
}

/* On very small screens (<576px), stack them 1 per row. 
   Let them expand in height as needed. */
@media (max-width: 576px) {
  .info-card {
    width: 100%;
    height: auto;
  }
}
Important:

If you still see them stacking vertically on large screens, ensure there’s no extra container or CSS from previous attempts that is overriding display: flex; or forcing width: 100% on each card.

Wrapping Up
With this setup:

You get four square-like cards in a single row on desktops.
They gracefully wrap to 2×2 or full-width on mobile.
Email will wrap onto multiple lines.
The Order Status is in a small “card” row beneath, so it’s no longer “ugly.”
Make sure to remove old styling (like .order-info { display:block }) that forces vertical stacking.
This should get you the horizontal, square info-card layout you want, plus a cleaner status updater section. Enjoy!
