<script>
function showFullSizePreview(src) {
    const modal = document.getElementById('previewModal');
    const modalImg = document.getElementById('modalImage');
    modal.style.display = 'flex';
    modalImg.src = src;

    modal.onclick = function(e) {
        if (e.target === modal || e.target.className === 'close-modal') {
            modal.style.display = 'none';
        }
    };
}
</script>

2) Additional CSS
Here’s some lightweight CSS to give the info cards a neat look, and style the status card as well. Adjust the colors, shadows, and spacing as you like:

css
Copy
/* Info Cards Layout */
.order-info-cards {
  /* uses d-flex flex-wrap gap-3 from Bootstrap classes */
}

/* Each card in "Order Information" */
.info-card {
  background: var(--surface-color); /* usually #fff */
  border: 1px solid var(--border-color);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
  padding: 1rem;
  width: 160px;          /* or flex: 1 1 auto, if you want them to expand */
  text-align: center;
  position: relative;
  transition: transform 0.2s ease;
}

/* Slight hover effect */
.info-card:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-md);
}

/* A prominent icon at the top */
.card-icon {
  font-size: 1.5rem;
  margin-bottom: 0.5rem;
  color: var(--primary-color);
}

/* The label & value */
.info-label {
  font-size: 0.8rem;
  color: var(--text-secondary);
  margin-bottom: 0.25rem;
}
.info-value {
  font-size: 1rem;
  font-weight: 600;
}

/* The "status card" on the right column */
.status-card {
  background: var(--surface-color);
  border: 1px solid var(--border-color);
  border-radius: var(--radius-md);
  box-shadow: var(--shadow-sm);
}

/* A bit more spacing on the inside */
.status-card form {
  padding: 1rem 0.5rem;
}

/* Freed up from “card style,” so no big container needed around .row */
.order-info-row {
  margin-top: 1rem;
}
Key Style Points
Info Cards: Each .info-card is a small “tile” with a border, a subtle shadow, and an icon at the top.
Status Card: The .status-card is a separate box with its own padding. You could add a heading or progress bar if you want more flair.
Icons: We used Font Awesome for the main icons (fa-info-circle, fa-calendar-alt, fa-tasks, etc.). Swap in any icons you like.
Responsive: On smaller screens, .col-md-8 and .col-md-4 will stack vertically, and the “cards” inside .info-card will wrap.
3) Final Touches / Variations
You might want a slightly different background for .info-card (like #f9fafb) to differentiate from the page.
For the “Order Status,” you could add a stepper or progress bar indicating the status visually (e.g., Pending → Processing → Completed).
You can also shrink or enlarge each info card’s width, or let them auto-fill the row (flex: 1 1 auto).
Enjoy experimenting with these ideas to get the perfect balance of visual appeal and clarity for your order page!
